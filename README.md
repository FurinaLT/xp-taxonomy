# xp-taxonomy

XP 词条分类项目第一期

主站进：https://furinalt.github.io/xp-taxonomy/site/

# 前言

在相当长的一段时间里，XP 类词汇都处在一种“被不断创造，却从未被系统整理”的状态中。

它们诞生得很快：来自亚文化、论坛、匿名社区、二次创作、玩笑、误解，甚至只是某一次偶然的命名；  
它们传播得也很快：被复用、被变形、被缩写、被重新解释，在不同圈层里拥有完全不同的语义重量。  
可与此同时，它们几乎从未被认真地**放在同一语境下进行系统性的审视**。

于是我们看到的是一种熟悉又混乱的景象：  
同一个词条，在不同地方指代不同的偏好；  
不同的词条，实际上描述着高度重叠的心理结构；  
还有大量“看似是 XP，其实更像是表现形式 / 叙事母题 / 情绪标签”的词，被混杂地使用着。

这些词汇往往在使用中先于定义，在传播中先于理解，  
久而久之，便形成了一套缺乏边界、却又被频繁引用的词汇体系。

---

正是在这样的背景下，这个 XP 分类项目被发起了。

本项目并不试图裁决“什么才是正确的 XP”，  
也不试图用道德、医学或意识形态去覆盖这些词条本身。  
它更倾向于一次**结构性的整理尝试**：  
将已经存在的词汇进行收集、拆解其语义构成，对比它们之间的异同，并尝试建立一个**相对稳定、可扩展的分类框架**。

这个框架并不是为了限制表达，  
而是希望在信息高度碎片化的现状下，  
至少提供一种可供参考的秩序。

---

然而，整理的过程本身并不轻松。

XP 类词汇的第一个困难，在于**概括的困难**。  
许多词条并非来源于清晰定义，而是来自体验、情绪、隐喻，甚至只在特定语境中成立的默契用法。  
它们很难被一句话准确描述：  
过于简化会导致误解，  
而过度展开又会迅速偏向个人叙述。

第二个困难，是**分类本身的困难**。  
XP 往往并非单一维度的概念，  
而是同时涉及角色关系、权力结构、身体意象、情绪需求、叙事视角等多个层面。  
任何试图将其纳入固定结构的行为，都不可避免地需要取舍与妥协。

此外，词汇本身并不是静止的。  
新的词条仍在不断产生，旧的词不断被重新命名、再解释、再包装；  
在网络文化高度加速的环境中，任何一次整理，都天然带有“滞后”的属性。

---

在个人层面，这些问题表现得更加直接。

目前，本项目的大部分词条收集、整理与分类，仍主要依赖人工完成；  
在信息分散、语义模糊、理解成本高的前提下，进度自然缓慢而吃力。  
有时，一个看似简单的词条，需要反复查阅来源、比对语境，才能勉强确认其核心含义；  
有时，已经整理出的结构，也会因为新词或新用法的出现而被迫调整，甚至推翻重来。

如果你在阅读过程中感受到不完整、不统一，甚至是混乱，  
那并非刻意为之，  
而更像是这一领域当前状态的真实反映。

---

即便如此，这个项目仍然选择继续推进。

XP 词汇本身就不太可能拥有一个“最终形态”，  
但只要有人愿意停下来，将这些零散的概念记录下来、整理出来，并放入同一框架中进行讨论，  
它们就不至于永远停留在碎片化的状态。

这，正是这个项目存在的理由。

## 修改

与 V0.2 不同的是，**保留了基本词条，去除了复合词条**，并把基本词条修改为**词条**，增加了**附属词**用于衡量、分类和判断词条所包含的附属词有哪些。

例如 **黑瞳** 这个词的意思包含 **黑色、瞳孔颜色**，再次拆分和合并可以获得 **颜色、瞳孔** 这些词，一个是表现词另一个是部位词。表现词权重更大（主观），所以把他放到 外貌与视觉特征 而不是 本体/生物结构。

## 编码

本项目采用 ICD-11 的编码方案进行词条的编码，具体格式如下：
- 第一位、第三位和第四位（第一级、第三级和第四级）为数字和大写字母组合
- 第二位（第二级）为大写字母
- 第五位和第六位（第五级和第六级）在第四位后加小数点，主体为数字和大写字母组合
