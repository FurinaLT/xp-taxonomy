## 描述

颅骨或面骨骨折是指由于外部直接或间接暴力作用于头部，导致颅骨或面部骨骼的连续性中断。这种损伤可表现为线性（骨折线未穿透整个颅骨厚度）、凹陷性（骨折部位向内塌陷）或粉碎性（骨骼碎裂成多块），常伴随硬脑膜、脑组织或面部软组织的继发性损伤。解剖范围涵盖颅盖骨、颅底骨以及眼眶、颧骨、鼻骨等面骨结构。[^1]
