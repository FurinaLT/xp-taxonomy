 **婚纱**是一件在婚礼或是婚宴时(凖)新娘穿着的所穿着打扮的漂亮、使人值得注意的西式婚服。它可指身上穿的纯白连身裙或长裙(长袍)，是外国结婚仪式时新娘所穿的结婚服饰。[^1]
