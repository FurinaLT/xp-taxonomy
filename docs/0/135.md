## 描述[^1]

这段关系基本上对[他们的正式伴侣](https://dic.pixiv.net/a/%E6%9C%AC%E5%91%BD)[保密](https://dic.pixiv.net/a/%E7%A7%98%E5%AF%86) 。在[婚姻](https://dic.pixiv.net/a/%E7%B5%90%E5%A9%9A)中的婚外情案例中，配偶通常会同意婚外情，但也存在婚外情一方欺骗另一方的情况。

[这](https://dic.pixiv.net/a/%E4%BA%8C%E8%82%A1)也被称为出轨，而且婚外情的数量会随着伴侣数量的增加而增加。然而，一个人的价值或人际关系的质量并不能仅仅用婚外情的数量来衡量。
