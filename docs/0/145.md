这种特殊类型的植入物的特点是**上下身分离，** 即使在被称为“ 植入式”植入物的类别中也是如此。

**关键要素是无法检查自己下半身状态的挫败感** 、只有臀部暴露的**超现实感** ，以及侵犯下半身（从脸上看不到）所带来的**恋物癖式的越轨感** 。有些作品故意省略对臀部的描绘，而只关注上半身的反应。
