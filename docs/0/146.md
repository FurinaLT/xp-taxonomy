## 描述[^1]

!!! quote inline end "参考"

无论是指植物还是动物，这都是指利用现有[个体的](https://dic.pixiv.net/a/%E5%80%8B%E4%BD%93)遗传信息，创造出另一个具有完全相同遗传信息的个体的过程。这个词源于[希腊语，](https://dic.pixiv.net/a/%E3%82%AE%E3%83%AA%E3%82%B7%E3%83%A3%E8%AA%9E) 意思是“切割”。自然[无性繁殖](https://dic.pixiv.net/a/%E7%84%A1%E6%80%A7%E7%94%9F%E6%AE%96)是指单个个体利用[体细胞而非生殖细胞](https://dic.pixiv.net/a/%E7%94%9F%E6%AE%96%E7%B4%B0%E8%83%9E)形成新个体的过程 [。](https://dic.pixiv.net/a/%E4%BD%93%E7%B4%B0%E8%83%9E) 在许多[植物](https://dic.pixiv.net/a/%E6%A4%8D%E7%89%A9)中，通过扦插等方法很容易从体细胞组织进行克隆，但在大多数[动物](https://dic.pixiv.net/a/%E5%8B%95%E7%89%A9)中则很困难。

然而，个体的特征并非完全由基因信息决定，因此“完全相同的基因信息”并不意味着“完全相同的特征”。此外，克隆体也不可能继承被克隆者的记忆。 **即使拥有相同的基因信息，克隆体和被克隆者也是完全不同的两个人** 。
