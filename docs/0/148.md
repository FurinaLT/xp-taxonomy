在穿着遮盖大部分乳房的衣物的情况下进行乳交行为。必须完全遮盖乳头和乳晕。[^1]

这是指穿着衣服进行口交的行为。即使只是简单地提到“ [服装](https://dic.pixiv.net/a/%E7%9D%80%E8%A1%A3) ”，不同插画家的描绘也各不相同。[^2]

- 把它压在你的衣服上
- 将[阴茎](https://dic.pixiv.net/a/%E7%94%B7%E6%80%A7%E5%99%A8)插入衣服下，夹在[双乳](https://dic.pixiv.net/a/%E3%81%8A%E3%81%A3%E3%81%B1%E3%81%84)之间
- 他解开她的衣服，将[阴茎](https://dic.pixiv.net/a/%E7%94%B7%E6%80%A7%E5%99%A8)插入她乳房的中间（这种技巧被称为“ [垂直乳交](https://dic.pixiv.net/a/%E7%B8%A6%E3%83%91%E3%82%A4%E3%82%BA%E3%83%AA) ”）。
- [泳衣](https://dic.pixiv.net/a/%E6%B0%B4%E7%9D%80)穿着[口交](https://dic.pixiv.net/a/%E3%83%91%E3%82%A4%E3%82%BA%E3%83%AA)做
