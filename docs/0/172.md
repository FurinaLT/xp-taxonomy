侧乳摩擦是一种[隔着衣物进行的乳房摩擦](https://dic.pixiv.net/a/%E7%9D%80%E8%A1%A3%E3%83%91%E3%82%A4%E3%82%BA%E3%83%AA) 。它指的是在男性仍穿着衣物的情况下，用乳房隔着衣物刺激其[生殖器](https://dic.pixiv.net/a/%E3%83%91%E3%82%A4%E3%82%BA%E3%83%AA) 。这种情况通常发生在隔着学校泳衣或比赛泳衣的布料进行摩擦时。[^1]
