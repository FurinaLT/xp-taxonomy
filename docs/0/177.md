“射精控制”一词有时指的是有意识地释放精液的行为。这表明射精也可能在不受控制的情况下无意识发生。这种射精被称为[遗精](https://dic.pixiv.net/a/%E9%81%BA%E7%B2%BE) ，这种现象在双性人中比在男性中更为常见。[^1]
