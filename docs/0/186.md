## 描述

异食癖是一种进食障碍，其特征是患者经常性地、持续性地食用非营养性的物质，例如泥土、粉笔、塑料、金属、纸张等。这种行为不仅对健康构成潜在威胁，还可能造成功能受损。异食癖通常发生在能够区分食物和非食物的个体中（大约2岁以上的儿童或成人），且这种行为需达到需要临床关注的程度。根据ICD-11，异食癖被归类在精神、行为或神经发育障碍下的喂食或进食障碍章节中，编码为6B84。[^1]
