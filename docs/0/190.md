在英语国家，它被称为“ [性爱机器](https://dic.pixiv.net/a/%E3%82%BB%E3%83%83%E3%82%AF%E3%82%B9%E3%83%9E%E3%82%B7%E3%83%BC%E3%83%B3) ”或“性交机器”，是该设备、 [性交](https://dic.pixiv.net/a/%E6%80%A7%E4%BA%A4)或其他性活动（如肛交或口交）的通用术语。

它被归类为一种捆绑 SM 游戏。实际上，这些装置通常只有[活塞式机械](https://dic.pixiv.net/a/%E3%83%94%E3%82%B9%E3%83%88%E3%83%B3%E3%83%9E%E3%82%B7%E3%83%B3)那么大，但在虚构作品中，也出现了具有各种功能的大型装置。

由于伴侣是[机器](https://dic.pixiv.net/a/%E6%A9%9F%E6%A2%B0) ，这种行为带有类似[凌辱（ryona）](https://dic.pixiv.net/a/%E3%83%AA%E3%83%A7%E3%83%8A) 的含义，涉及性实验 、 训练和折磨 ，而非生殖行为。它也像一个大型[成人玩具](https://dic.pixiv.net/a/%E5%A4%A7%E4%BA%BA%E3%81%AE%E3%81%8A%E3%82%82%E3%81%A1%E3%82%83) ，旨在通过无休止的折磨使伴侣获得极致的[快感](https://dic.pixiv.net/a/%E5%BF%AB%E6%A5%BD) 。
