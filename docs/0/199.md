**兽娘**所具有的动物要素主要为“兽耳”和“兽尾”，也有在此基础上拥有“兽爪”和“兽足”的类型。但是基本上拥有人的面孔和身躯，相比兽（兽人）而言动物要素较少，相对地受众也比较广泛。如果说“兽”是对动物的拟人化的话，“兽娘”给人的感觉更像是“女性的拟兽化”。[^1]
