## 描述[^1]

正如开头所提到的，这些是“带有性意图画在身体表面的图案”。

最常见的纹身类型可能是在女性下腹部、 [子宫](https://dic.pixiv.net/a/%E5%AD%90%E5%AE%AE)上方的位置。这种图案位于这个位置本身就暗示着暴露内部生殖器的意图，因此它通常被认为是一种情色标记。

同样，这些纹身有时会纹在臀部和胸部等具有性暗示的部位。此外，一些大型图案还会延伸到身体的其他部位，例如手和脚。然而，即使纹在性暗示较少的部位，如果满足其他条件，也可能被认为是淫秽的。

虽然形状各异，但经常会描绘心形符号或类似图案。绘制在下腹部的图案尤其常被解读为子宫的象征，而且通常会包含阴道、子宫、输卵管或卵巢的程式化图像。
