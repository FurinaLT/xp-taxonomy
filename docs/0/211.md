子宫损伤是指由于外力、医源性操作或病理因素导致的子宫解剖结构或生理功能的异常改变，可能累及子宫壁全层（浆膜层、肌层、子宫内膜）、子宫颈或毗邻血管。根据损伤机制可分为开放性损伤（如穿透性创伤）或闭合性损伤（如钝性挫伤），部分病例可能并发出血、感染或器官功能障碍。[^1]
