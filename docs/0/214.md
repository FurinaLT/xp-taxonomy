一种设计用于[睡眠](https://dic.pixiv.net/a/%E7%9C%A0%E3%82%8B)时拥抱的[枕头](https://dic.pixiv.net/a/%E6%9E%95) ，也称为抱枕。该产品着眼于人们拥抱柔软物体时会感到安全这一事实，创造了一种类似[同床共眠的](https://dic.pixiv.net/a/%E6%B7%BB%E3%81%84%E5%AF%9D)体验，以促进安稳睡眠。[^1]
