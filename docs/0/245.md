从字面上讲，这是将一个人的[个性](https://dic.pixiv.net/a/%E4%BA%BA%E6%A0%BC) （ [记忆](https://dic.pixiv.net/a/%E8%A8%98%E6%86%B6) 、 [知识](https://dic.pixiv.net/a/%E7%9F%A5%E8%AD%98)等）转化为[粪便](https://dic.pixiv.net/a/%E6%8E%92%E6%B3%84%E7%89%A9)并喷洒出去的行为。此前， [莫尔杰索化过程](https://dic.pixiv.net/a/%E3%83%A2%E3%83%AB%E3%82%B2%E3%83%83%E3%82%BD%E3%83%A8%E5%8C%96)的一个副产品被称为“ [个性射精](https://dic.pixiv.net/a/%E4%BA%BA%E6%A0%BC%E5%B0%84%E7%B2%BE) ”，在这个过程中，一个人的个性会以精液的形式排出体外。[^1]
