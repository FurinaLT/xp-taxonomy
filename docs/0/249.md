“公开处决”一词的本义是指在公众面前处死[死刑犯](https://dic.pixiv.net/a/%E6%AD%BB%E5%88%91%E5%9B%9A) 。虽然行刑通常是为了杀鸡儆猴，但吸引大量民众围观也并不罕见。例如，英国的泰伯恩庄园曾长期作为公开处决场所。据说，当年著名的越狱犯杰克·谢泼德在那里被处决时，成千上万的人涌来，只为一睹他的真容。[^1]
