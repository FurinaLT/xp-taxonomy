这是一种[死刑执行](https://dic.pixiv.net/a/%E5%87%A6%E5%88%91)方式，用[斧头](https://dic.pixiv.net/a/%E6%96%A7) 、 [砍刀](https://dic.pixiv.net/a/%E9%89%88) 、 [剑](https://dic.pixiv.net/a/%E5%88%80)或[断头台](https://dic.pixiv.net/a/%E3%82%AE%E3%83%AD%E3%83%81%E3%83%B3)等重型[刀刃武器](https://dic.pixiv.net/a/%E5%88%83%E7%89%A9)砍下死刑犯的[头颅](https://dic.pixiv.net/a/%E9%A6%96) ，使其流血而死。由于这种方法被认为能立即致死，因此常被用于处决贵族，尤其是在[西方国家](https://dic.pixiv.net/a/%E8%A5%BF%E6%B4%8B) 。[^1]
