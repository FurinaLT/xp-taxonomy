“男士泳装挑战”是一种涉及[异](https://dic.pixiv.net/a/%E7%94%B7%E8%A3%85)装癖、 [暴露癖](https://dic.pixiv.net/a/%E9%9C%B2%E5%87%BA%E3%83%97%E3%83%AC%E3%82%A4)和[羞辱游戏的](https://dic.pixiv.net/a/%E7%BE%9E%E6%81%A5%E3%83%97%E3%83%AC%E3%82%A4)场景。

虽然“男士泳装挑战”一词可以广泛地用来描述穿着男士泳装的挑衅行为，但它主要**指的是一种特定情况，即“一个性特征不明显的女孩穿着男士泳装（ [泳裤](https://dic.pixiv.net/a/%E6%B5%B7%E3%83%91%E3%83%B3) 、冲浪短裤等），上身[赤裸地](https://dic.pixiv.net/a/%E3%83%88%E3%83%83%E3%83%97%E3%83%AC%E3%82%B9)前往游泳池或海滩等地，伪装成[男人](https://dic.pixiv.net/a/%E7%94%B7%E8%A3%85)以避免被认出是女人** ”。
