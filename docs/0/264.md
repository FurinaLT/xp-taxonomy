这是一种将身体转化为[精液的](https://dic.pixiv.net/a/%E7%B2%BE%E6%B6%B2)变身类型。几乎所有这类作品都属于 R-18 级别，通常描绘角色被巨大的[男性器官](https://dic.pixiv.net/a/%E7%94%B7%E6%80%A7%E5%99%A8)吸收，变成精液，然后[被射精](https://dic.pixiv.net/a/%E5%B0%84%E7%B2%BE) 。因此，在 Pixiv 上，这类作品有时会被贴上 [“扶他那利”](https://dic.pixiv.net/a/%E3%81%B5%E3%81%9F%E3%81%AA%E3%82%8A) 或 [“吞噬一切](https://dic.pixiv.net/a/%E4%B8%B8%E5%91%91%E3%81%BF) ”之类的标签。[^1]
