“颈阴茎”是指男性生殖器从连接头部和躯干的颈部（ [子宫颈](https://dic.pixiv.net/a/%E9%A0%B8) ）或男性生殖器本身生长出来的一种情况。其显著特征是头部和躯干可以分离，就像[无头骑士](https://dic.pixiv.net/a/%E3%83%87%E3%83%A5%E3%83%A9%E3%83%8F%E3%83%B3)一样。无论男性生殖器是从头部还是躯干长出，在本文中，它们都将统称为“颈阴茎”。[^1]

[^1]: [首チンコ (くびちんこ)とは【ピクシブ百科事典】](https://dic.pixiv.net/a/%E9%A6%96%E3%83%81%E3%83%B3%E3%82%B3). 2026-06-2
