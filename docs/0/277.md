[单恋](https://dic.pixiv.net/a/%E7%89%87%E6%81%8B) ：当两个人中只有一方单方面爱着另一方时。

在[恋爱漫画](https://dic.pixiv.net/a/%E6%81%8B%E6%84%9B%E6%BC%AB%E7%94%BB)中，这通常以酸甜苦辣的方式描绘，但在以[个人](https://dic.pixiv.net/a/%E5%8D%98%E4%BD%93%E8%90%8C%E3%81%88)角色为中心的漫画中，这通常被描绘成一种角色塑造形式（无论恋爱是否成功，它都揭示了角色的喜好，这可能会让[旁观者](https://dic.pixiv.net/a/%E7%AC%AC%E4%B8%89%E8%80%85)觉得有趣）。[^1]
