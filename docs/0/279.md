心碎指的是对所爱之人单相思却得不到回应，或是[某种浪漫关系](https://dic.pixiv.net/a/%E6%81%8B%E6%84%9B)的终结。广义上讲，它也可以指处于社会地位差异很大的人（例如名人）的单相思。然而，配偶[的去世](https://dic.pixiv.net/a/%E6%AD%BB%E5%88%A5)并不属于心碎的范畴。此外，狭义上讲，恋人之间的[分离](https://dic.pixiv.net/a/%E9%9B%A2%E5%88%A5)也不包括在内。它的反义词是“ [成功的爱情](https://dic.pixiv.net/a/%E5%BE%97%E6%81%8B) ”或“ [浪漫的圆满](https://dic.pixiv.net/a/%E6%81%8B%E6%84%9B%E6%88%90%E5%B0%B1) ”。[^1]
