**幼女**即幼小的女孩。以人类来说，一般指的是介于[婴儿](https://mzh.moegirl.org.cn/%E5%AE%9D%E5%AE%9D "宝宝")与少女之间的年龄段。幼女的年龄界定没有严格的定义。即使是粗略的定义，从法律上看和从生理学上看，也有着不同的定义范围。[^1]

[^1]: [幼女 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E5%B9%BC%E5%A5%B3). 2026-06-25
