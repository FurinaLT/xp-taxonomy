骑乘体位的变体。无论是否带有浪漫的意味 ，它指的是手牵手时采用的骑乘体位。

虽然“手牵手面对面骑乘体位 ”很常见，但“手牵手后入式骑乘体位 ”在相当多的成人内容作品中也能找到。“手牵手面对面骑乘姿势”与“ Ori-cha-usu ”姿势相同，是日本自江户时代以来就为人所知的性爱姿势之一（ 48 种隐秘姿势之一）。
