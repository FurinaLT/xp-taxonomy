一种面对面坐着进行的体位。其特点是身体接触程度很高。此外，由于插入方臀部较高，接受方的位置也较高，因此更容易进行[哺乳](https://dic.pixiv.net/a/%E4%B9%B3%E5%90%B8%E3%81%84)和[吸吮](https://dic.pixiv.net/a/%E6%8E%88%E4%B9%B3)等在其他体位中难以同时进行的动作。[^1]
