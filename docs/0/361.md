感官遮蔽有两种类别：[^1]

- 药物或超自然能力干预：这种模式涉及通过[麻醉](https://dic.pixiv.net/a/%E9%BA%BB%E9%85%94) 、 [魔法](https://dic.pixiv.net/a/%E9%AD%94%E6%B3%95)或[改造来](https://dic.pixiv.net/a/%E6%94%B9%E9%80%A0)阻断[一个人的触觉](https://dic.pixiv.net/a/%E8%A7%A6%E8%A6%9A)和[性感觉](https://dic.pixiv.net/a/%E6%80%A7%E6%84%9F) ，导致这个人完全没有意识或无法注意到自己的身体随意遭受任何性行为。
- 物理干预：这种模式包括通过[蒙眼](https://dic.pixiv.net/a/%E7%9B%AE%E9%9A%A0%E3%81%97) 、 [全头面罩](https://dic.pixiv.net/a/%E5%85%A8%E9%A0%AD%E3%83%9E%E3%82%B9%E3%82%AF)和[耳塞](https://dic.pixiv.net/a/%E8%80%B3%E6%A0%93)等方法来阻挡[视觉](https://dic.pixiv.net/a/%E8%A6%96%E8%A6%9A)和[听觉](https://dic.pixiv.net/a/%E8%81%B4%E8%A6%9A) 。
