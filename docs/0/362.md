## 描述[^1]

正太指的是受[正太控（](https://dic.pixiv.net/a/%E3%82%B7%E3%83%A7%E3%82%BF%E3%82%B3%E3%83%B3) 指喜欢年轻男孩的人群）喜爱的可爱年轻男孩（角色） 。它是[萝莉](https://dic.pixiv.net/a/%E3%83%AD%E3%83%AA) （ [萝莉角色](https://dic.pixiv.net/a/%E3%83%AD%E3%83%AA%E3%82%AD%E3%83%A3%E3%83%A9) ）的男性对应词。此外，为了与萝莉区分开来，正太通常被描绘得比萝莉更具男性气质。

然而，风格和其他因素的差异实在太大 **，无法简单地按年龄进行分类。尤其需要注意的是，每个人的喜好都不同，“正太”的年龄和外貌范围也因人而异。** 举个极端的例子， [鸦天狗](https://dic.pixiv.net/a/%E5%8D%83%E7%BE%BD%E9%BB%92%E4%B9%83)甚至认为中山金肉君也是正太。这足以说明“正太”的定义因人而异。
