纳差，又称食欲减退或食欲不振，是指病理性摄食欲望降低，表现为主动进食意愿减弱和摄入量减少。需通过患者主诉结合病史采集、体格检查及实验室/影像学检查排除器质性疾病与功能性紊乱。[^1]

此词条描述的是一种：体重过轻，以至于皮肤凹陷，或者骨骼结构的任何部分贴合皮肤轮廓[^2]的情况。

[^2]: [anorexic - EHWiki](https://ehwiki.org/wiki/anorexic). 2026-06-20
