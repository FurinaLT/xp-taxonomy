## 描述

身体上至少有一个嘴巴出现在意想不到的位置（例如手、躯干、尾巴）。[^1]

除了主要在胃部开口的腹部型之外，还有在全身各处有多个开口的类型，以及在手掌和后脑勺有开口的类型。

腹口的情况多种多样，例如“没有肚脐 ，取而代之的是嘴”，以及“肚脐本身具有捕食功能”（ 食脐动物 ），还有“通常模仿肚脐，但当它显露出真面目时，嘴就会出现”。这些情况有时被称为“ 脐口 ”。然而，反过来，也存在“嘴位于与肚脐不同的位置，并且两者都存在”的腹口。[^2]
