皮肤良性黑素细胞瘤（Benign Melanocytic Naevus），又称黑素细胞痣或痣，是由皮肤黑素细胞良性增殖形成的局限性病变。这些病变由分化成熟的黑素细胞组成，表现为皮肤上颜色均匀的斑疹、丘疹或结节。根据发生时间可分为先天性和后天性两类，后天性痣通常在儿童期至青年期逐渐显现。[^1]
