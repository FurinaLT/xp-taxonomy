## 描述

参考自 ICD11 的 09 视觉系统疾病，描述为 这是指视觉系统的任何疾病，包括眼睛和附件、视路和大脑区域，它们启动和控制视觉感知和视觉引导行为。[^1]

[^1]: [用于死因与疾病统计的ICD-11](https://icd.who.int/browse/2026-01/mms/zh#868865918). 2026-06-21
