## 简介

!!! quote inline end "图片参考"
	<div align="center">
		<img src='https://storage.moegirl.org.cn/moegirl/commons/8/8b/%E5%A0%95%E5%A4%A9%E4%BD%BF.jpg' alt='Image' style='max-width: 300; max-height: 400;'>
	</div>

堕天使就是因故被褫夺（丧失）或放弃天使的身份、遭天神逐出或志愿离开[天堂](https://mzh.moegirl.org.cn/%E5%A4%A9%E5%A0%82 "天堂")的天使。永久离开天堂的天使，轻则降落凡间，用普通人的身份渡过一生，严重者坠入魔道成为黑暗势力（包含变成[恶魔](https://mzh.moegirl.org.cn/%E6%81%B6%E9%AD%94\(%E5%8E%9F%E5%9E%8B\) "恶魔(原型)")）的一分子。许多作品里的堕天使通常是描绘成为黑暗势力的后者居多。[^1]

[^1]: [堕天使 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E5%A0%95%E5%A4%A9%E4%BD%BF). 2026-02-09
