## 简介

!!! quote inline end "画作参考"
	- [首チンコ わたしが首チンコになれるわけないじゃん、ムリムリ！ - FBCL的插画 - pixiv](https://www.pixiv.net/artworks/140007081)

**阴茎化**是指人类女性或雌雄同体女性的形状或状态发生变化，其中整个身体或身体的一部分变得像阴茎一样，或者被阴茎吸收并与之融合。[^1]

这是用来形容女性或女性扶她将整个身体或身体的一部分变成阴茎的表达方式。目标可能会独立变形，也可能会被阴茎吸收并与之融合。
