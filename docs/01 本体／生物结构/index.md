**本体／生物结构** 包含 生物种类，例如人类、血族、龙族等，指不同的生物种族。

**身体结构**，例如翅膀、角、多眼等，指身体部位 延伸出的新结构或变异(在原有形状、颜色等性质上更改)成的不同结构，或现实中不存在的身体结构。
- **简单型变异**指在原有结构基础上进行形状、颜色的改变过程，原有结构不变，例如红瞳、白毛等，多数可归类于 04 外貌与视觉特征。
- **基础型变异**指在原有结构上进行生长、改变的过程，或指对应生物身体上组织的生长、改变的物质属于该生物任意部位的组织材质、物质，例如巨乳、乳t阴j等
- **改变型变异**跟基础生长型不同的点在于增长物质不跟随生长位置的材质，或不属于对应生物身体上任意部位的组织材质、物质，比如 猫耳、龙角 等。

!!! tip
    这里的“**变异**”不是现实中存在的变异，而是 ACGN 文化中虚构、架空的一种非现实的想象。

**收录条件/范围**：生物种类选择类人，或类人延伸的物种，范围在 XP 一词定义之内，比如 哥布林、水猴子、河童 等物种不选入内（因为没人的 XP 是哥布林什么的(应该吧) ）；收录的词条的选择可能会很主观吧..。

已被移除的：物理状态，例如不死、不老、半透明等，指在物理状态下的 架空、现实不存在 的情况。
