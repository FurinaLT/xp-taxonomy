## 简介

**睫毛精**是对拥有外观特殊的睫毛的角色的一种褒义的戏称，也是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")**。

睫毛具有防尘、保护眼睛的作用，较长的睫毛能显得[眼睛更大](https://mzh.moegirl.org.cn/%E5%A4%A7%E7%9C%BC%E7%9D%9B "大眼睛")，变相地提升[颜值](https://mzh.moegirl.org.cn/%E9%A2%9C%E5%80%BC "颜值")。而睫毛异常地长、异常地厚，或外观非常特殊的情况下，又会为角色赋予另类、夸张的第一印象。[^1]
