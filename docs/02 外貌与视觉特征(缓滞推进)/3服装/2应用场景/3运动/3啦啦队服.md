## 简介

**啦啦队服**是啦啦队穿着的服装。

因为啦啦队员会在体育比赛期间进行各种表演，其中不乏青春靓丽的可爱小姐姐，所以啦啦队也是体育比赛中一道靓丽的风景线。

<del>当然，也是很多趣味怪异的死宅们幻想对象的服装之一。</del>[^1]
