## 简介

**民族风格**是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**萌属性**之一。

民族风格一般指“民族风格的服饰”的简称，每个国家都有自己的民族服饰。

中国的“民族风格的服饰”包括国内少数民族的服饰。包括苗族的传统银饰、维吾尔族的艾得丽斯连衣裙等服装，也包括汉服等汉族传统服饰。[^1]
