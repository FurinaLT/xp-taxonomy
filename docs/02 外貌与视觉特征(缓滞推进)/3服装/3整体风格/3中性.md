## 简介

**中性**是中性化的服饰和妆容的简称，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的萌属性。

不管是在现实生活中还是在二次元中，“中性”一词所指代的为“女生穿偏男性化的服装”的频率要比“男生穿偏女性化的服装”的频率要高。这是因为社会上对男性女装的接受度没有女性男装的接受度高。[^1]
