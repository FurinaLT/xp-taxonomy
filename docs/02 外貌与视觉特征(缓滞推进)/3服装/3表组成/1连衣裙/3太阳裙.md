## 简介

**太阳裙**（英语：**sundress**；日语：**サンドレス**）是一种在温暖或者酷热的[时节](https://mzh.moegirl.org.cn/%E5%AD%A3%E8%8A%82 "季节")穿着的**[连衣裙](https://mzh.moegirl.org.cn/%E8%BF%9E%E8%A1%A3%E8%A3%99 "连衣裙")**，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**萌属性**。

**太阳裙**（英语：**Sundress**），也译**肩带式连衣裙**，**吊带连衣裙**，**吊带背心裙**。是一种在温暖或者酷热的时节穿着的[连衣裙](https://mzh.moegirl.org.cn/%E8%BF%9E%E8%A1%A3%E8%A3%99 "连衣裙")。通常情况下，这种服装由轻薄面料（一般为纯棉）制成，通常整体很宽松，并作为休闲装穿着。这种服装领口较大，肩带较窄。太阳裙通常不搭配（无纽扣或者拉链的）外套穿着，也不像传统的[背心裙](https://mzh.moegirl.org.cn/%E6%97%A0%E8%A2%96%E8%BF%9E%E8%A1%A3%E8%A3%99 "无袖连衣裙")那样搭配诸如罩衫、毛衣或者T恤衫穿着。
