## 简介

**紧身衣**（英语：**body suit / tight clothing**；日语：**ボディスーツ/ぴっちりスーツ**）是一种通常包裹除头部以外身体的、紧贴身体的服装，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**萌属性**之一。

紧身衣的外观一般包括连体，且覆盖头部以下的全部身体（包括四肢和手），或者露出部分肢体。

穿着这类服装的角色能够进一步的表现出其身形。在部分科幻系[ACGN](https://mzh.moegirl.org.cn/ACGN "ACGN")作品中常有此类服装登场。[^1]
