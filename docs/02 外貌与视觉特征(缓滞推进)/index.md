## 分类解释

**外貌特征**：指一个人外表的特征形态，可以包含 `1B4` 里的组织形态。外貌特征是先天性的、固定型的表现，如果是因为情绪或是身体状态导致的外貌变化则分在 `03` 行为模式中。

!!! tip "什么是“固定型”的？"
	比如头型，不会因为情绪或身体形态导致突然变化的。如果按照这样的定义，那么死鱼眼可分为行为模式或者是外貌特征里，本wiki允许标签在两个或多个分类下同时存在。

**体型**：指人的身体整体的形态。

**饰品**：服装的子级，比单件衣物小，例如耳钉、发夹等。

### 服装分类解释

**服装**：是衣服鞋装饰品等的总称[^1]，饰品为它的子级，在本wiki中作为同级存在。

分类服装可以有两种方式来进行分类：应用场景和服装设计（细分）。应用场景表示穿着的场合、服装的功能等。服装设计则会更细，描述特定服装的细节、装饰等。两者都可以存在，在本wiki中使用融合的方法将两者保留，当然也会偏向应用场景方向一些。
