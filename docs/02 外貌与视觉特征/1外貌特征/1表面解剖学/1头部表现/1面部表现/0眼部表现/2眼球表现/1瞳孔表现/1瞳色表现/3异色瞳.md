## 简介

**异色瞳**，即两眼眸色不一致，又称“**双色瞳**”、“**阴阳瞳**”或“**阴阳双瞳**”，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")里的角色特征及[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")之一。

一般而言，现实中最容易有异色瞳的都是[动物](https://mzh.moegirl.org.cn/%E5%8A%A8%E7%89%A9 "动物")，例如[猫](https://mzh.moegirl.org.cn/%E7%8C%AB "猫")。其中，最著名的异色瞳猫种便是波斯猫和其混血变种临清狮猫。[^1]
