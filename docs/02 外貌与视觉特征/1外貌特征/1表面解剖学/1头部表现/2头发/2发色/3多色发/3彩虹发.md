## 简介

**彩虹发**（rainbow hair），是一种罕见的发色，也是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")**。

彩虹发属于极为罕见的发色，是指具像彩虹的色泽一样的七彩发色。~~但在[玛丽苏](https://mzh.moegirl.org.cn/%E7%8E%9B%E4%B8%BD%E8%8B%8F "玛丽苏")型角色的人设上面又极为常见。~~
