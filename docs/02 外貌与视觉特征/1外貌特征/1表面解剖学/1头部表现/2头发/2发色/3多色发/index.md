## 简介

**多色发**是一（多）种发色，也是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")**。

多色发指同时拥有大于两种发色的头发，通常在三种颜色到五种颜色之间，同时拥有七色头发的角色就属于[彩虹发](https://mzh.moegirl.org.cn/%E5%BD%A9%E8%99%B9%E5%8F%91 "彩虹发")了。[^1]
