**背心/马甲**（英语：**vest(西服背心.美) / waistcoat(西服背心.英) / singlet(运动背心) / underwaist(单背心)**；日语：**チョッキ/ベスト/シングレット(运动背心)**）是一类常见的服饰，有多种款式，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**萌属性**。

背心/马甲均是指一类无领无袖、且较短的上衣。马甲和背心在样式上相似但又有区别，或者说马甲是（广义）背心的一种。[^1]
