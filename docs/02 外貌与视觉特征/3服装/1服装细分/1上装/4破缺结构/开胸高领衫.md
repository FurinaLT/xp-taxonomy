**开胸高领衫**（英语：**keyhole turtleneck / open-chest turtleneck / that turtleneck**；日语：**胸開（むねあ）きタートルネック/例（れい）のタートルネック**）是一种于2014年末在2次元流行的一种**真实存在**的服装，其最大的特征是胸部大范围的开口。

开胸高领衫，是一种时尚冬季时装。该服装最大的特征是~~明明是保暖服装却做了一个~~胸前的大开口，使得女性的胸部上半以及乳沟（如果有）一览无余，凸显女性魅力的服装。[^1]
