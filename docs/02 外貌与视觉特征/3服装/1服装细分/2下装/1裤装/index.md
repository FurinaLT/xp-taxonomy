## 简介

**长裤**，是裤子中的一个大类，与[短裤](https://mzh.moegirl.org.cn/%E7%9F%AD%E8%A3%A4 "短裤")相对，字面意思即为“长度较长的裤子”，一般指的是从腰部延伸至及小腿下部或者脚踝的裤子。与长裤相对的概念有[短裤](https://mzh.moegirl.org.cn/%E7%9F%AD%E8%A3%A4 "短裤")、半腿裤等。[^1]

[^1]: [长裤 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E9%95%BF%E8%A3%A4)
