## 简介

**短裤**（英语：**shorts**；日语：**半（はん）ズボン/ショーツ**）是不过膝或及膝的裤类下装，是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的萌属性。

在ACG文化中，短裤似乎和[正太](https://mzh.moegirl.org.cn/%E6%AD%A3%E5%A4%AA "正太")绑定了起来。可可爱爱的小男孩穿着短裤卖萌耍帅，不需要台词，这身打扮本身就是萌点。二次元的女生中也有穿短裤的，特别是[中性](https://mzh.moegirl.org.cn/%E4%B8%AD%E6%80%A7 "中性")化的角色。短裤会让角色显得更活泼。如果是女性[偶像](https://mzh.moegirl.org.cn/%E5%81%B6%E5%83%8F\(%E8%90%8C%E5%B1%9E%E6%80%A7\) "偶像(萌属性)")经常穿短裤登台，说明[公式](https://mzh.moegirl.org.cn/%E5%85%AC%E5%BC%8F "公式")想重点塑造其“男孩子气”。[^1]

[^1]: [短裤 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E7%9F%AD%E8%A3%A4). 2026-03-14
