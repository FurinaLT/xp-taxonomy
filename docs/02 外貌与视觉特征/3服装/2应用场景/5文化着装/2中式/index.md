**中国风格**（英语：**Chinese style / Chinoiserie**；日语：**中華風（ちゅうかふう）**）。中国风格一般指“中国风格的服饰”的简称，包括但不限于 [旗袍](https://mzh.moegirl.org.cn/%E6%97%97%E8%A2%8D "旗袍")/[长衫](https://mzh.moegirl.org.cn/index.php?title=%E9%95%BF%E8%A1%AB&action=edit&redlink=1 "长衫（页面不存在）")、[袄裤](https://mzh.moegirl.org.cn/index.php?title=%E8%A2%84%E8%A3%A4&action=edit&redlink=1 "袄裤（页面不存在）")、[汉服](https://mzh.moegirl.org.cn/%E6%B1%89%E6%9C%8D "汉服")、[唐装](https://mzh.moegirl.org.cn/index.php?title=%E5%94%90%E8%A3%85&action=edit&redlink=1 "唐装（页面不存在）")、[中山服](https://mzh.moegirl.org.cn/index.php?title=%E4%B8%AD%E5%B1%B1%E6%9C%8D&action=edit&redlink=1 "中山服（页面不存在）")、[马褂](https://mzh.moegirl.org.cn/index.php?title=%E9%A9%AC%E8%A4%82&action=edit&redlink=1 "马褂（页面不存在）")等服装，并且往往以红色为主色调。进入新世纪，一些现代中国的元素也开始加入其中，比如[红领巾](https://mzh.moegirl.org.cn/%E7%BA%A2%E9%A2%86%E5%B7%BE "红领巾")、[解放帽](https://mzh.moegirl.org.cn/%E8%A7%A3%E6%94%BE%E5%B8%BD "解放帽")和熊猫主题的服装、饰品等。[^1]

[^1]: [中国风格 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E4%B8%AD%E5%9B%BD%E9%A3%8E%E6%A0%BC). 2026-02-22
