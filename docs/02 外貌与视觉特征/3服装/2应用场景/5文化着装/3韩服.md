**韩服**（韩语：한복／韓服）/**朝鲜服**（朝鲜语：조선옷／朝鮮옷）/**朝鲜族服饰**（中国朝鲜语：조선족 복식／朝鮮族 服飾）（英语：**hanbok / Korea dress**；日语：**韓服（かんふく）**/**朝鮮服（ちょうせんふく）**），是朝鲜族的传统服饰。是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的**萌属性**。日语中韩服与汉服发音一样，都是かんふく[^1]

[^1]: [韩服 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E9%9F%A9%E6%9C%8D). 2026-01-31
