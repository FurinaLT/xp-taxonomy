**布偶装**（英语：**mascot costume / mascot suit / fursuit(毛布偶装/兽装)** ；日语：**着（き）ぐるみ**/**縫（ぬ）いぐるみ**）指人体穿着用布偶，覆盖穿着者全身，为了扮演[人外](https://mzh.moegirl.org.cn/%E4%BA%BA%E5%A4%96 "人外")生物用的特殊衣物。

布偶装有多种类型，本条目将描述广义意义上的布偶装。

近年以来各种人为了戏剧或者单纯地喜欢布偶装的业余爱好者自己制作布偶装的有很多。[^1]

[^1]: [布偶装 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E5%B8%83%E5%81%B6%E8%A3%85). 2026-02-01
