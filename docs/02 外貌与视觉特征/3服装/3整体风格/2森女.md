## 简介

**森女**(森ガール、Mori Girl)是指崇尚简单的生活方式，打扮得像是从[森林](https://mzh.moegirl.org.cn/%E6%A3%AE%E6%9E%97 "森林")中走出来的女孩 ，喜欢自然舒适的物品，比一般人更加热爱大自然的舒适生活。也是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")。

“森ガール”一词最早出自日本社群网站“mixi”的森ガール群组，群组创始人Choco被评价为「森にいそうな格好だね」。这一群组在2006年8月开设，到2008年“森女”一词成为热门话题，到2009年时这一群组已经有三万五千人。[^1]

[^1]: [森女 - 萌娘百科 万物皆可萌的百科全书](https://mzh.moegirl.org.cn/%E6%A3%AE%E5%A5%B3). 2026-02-03
