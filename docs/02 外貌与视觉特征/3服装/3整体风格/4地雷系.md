## 简介

**地雷系**（英语：**Jirai Kei (fashion) / landmine style / landmine type**；日语：**地雷系（じらいけい）**）是日本社媒（SNS，Social Networking Service/Site）发展而来一种对性格类型的描述，后演化对一种特定类型形象的描述，也是[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")的一种。

“[地雷](https://mzh.moegirl.org.cn/%E5%9C%B0%E9%9B%B7\(%E6%AD%A6%E5%99%A8\) "地雷(武器)")”本指一种通过埋伏在地下的爆炸杀伤武器，放置后表面正常，待对象踏进目标范围后一下带来爆炸的严重后果。因此在日常词汇，也会引用此意，如中文网络流行词“踩雷”“雷区”“雷点”等，表示某行为实践后，发生与预期大为不同的负面结果等意。
