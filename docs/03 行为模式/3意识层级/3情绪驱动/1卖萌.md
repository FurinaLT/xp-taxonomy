## 简介

**卖萌**是网络用语。原本是ACG界词汇“[萌](https://mzh.moegirl.org.cn/%E8%90%8C "萌")”的引申词汇，即“刻意显示自身的萌”，后来在网络上意义又有所延伸。2011年度新浪微博盘点中十大年度网络语之一。也是[ACG](https://mzh.moegirl.org.cn/ACG "ACG")[次文化](https://mzh.moegirl.org.cn/%E6%AC%A1%E6%96%87%E5%8C%96 "次文化")中的一种[萌属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")。

卖萌的原意是显示自己的萌点可爱。为中性词。形容女性或孩子一般为褒义，其余视语境而定。

**卖萌的效果，与卖萌者的长相成正比。**[^1]
