## 简介

**恶意卖萌**是用于形容部分角色行为或[属性](https://mzh.moegirl.org.cn/%E8%90%8C%E5%B1%9E%E6%80%A7 "萌属性")的词语。

一开始指卖萌过度、过于刻意而不自然、表演痕迹太重、为了[卖萌](https://mzh.moegirl.org.cn/%E5%8D%96%E8%90%8C "卖萌")而卖萌等原因，导致看的人反感。

**但是演变到现在，这个词汇已偏向一种[黑色幽默](https://mzh.moegirl.org.cn/%E9%BB%91%E8%89%B2%E5%B9%BD%E9%BB%98 "黑色幽默")，常被用于戏谑和调侃。**
