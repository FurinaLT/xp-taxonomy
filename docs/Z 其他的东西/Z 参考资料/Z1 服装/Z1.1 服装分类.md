## 四、使用建议

### 如何记忆这294个属性？

```
按"部位+变化"逻辑分组：
1. 先记6大组（长度、版型、领型、袖型、图案、细节）
2. 每组内按"基础→变体"记忆：
   - 领型：圆领→V领→U领→方领→特殊领
   - 袖型：无袖→短袖→七分→长袖→特殊袖
3. 重点记20-30个最常用的，其余查表
```

### 实战应用公式

```
完整描述 = [品类] + [长度] + [版型] + [领型] + [袖型] + [图案] + [细节]

例：
Dress（品类）+ midi length（长度）+ a-line（版型） 
+ v-neck（领型）+ short sleeve（袖型） 
+ floral print（图案）+ ruffle detail（细节）
= 中长款A字V领短袖花卉印花荷叶边连衣裙
```
