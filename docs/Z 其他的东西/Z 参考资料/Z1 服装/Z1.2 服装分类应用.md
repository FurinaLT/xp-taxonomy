## 特殊服装分类解析

| **女仆装** | dress / top + skirt | 围裙+荷叶边+蝴蝶结+黑白配色+中长款 | **角色扮演变体** |

### 二、详细拆解每个服装

#### 1. 拉拉队服（Cheerleading Uniform）

```
基础品类：dress（连衣裙款）或 top + skirt（分体款）
├── 长度变体：short length（超短，便于活动）
├── 版型变体：a-line 或 fit and flare（活泼动感）
├── 袖型变体：sleeveless（无袖）或 short sleeve（短袖）
├── 颜色图案：color block（撞色）+ team logo（队标）
├── 细节装饰：pleated skirt（百褶裙）+ ruffle（荷叶边）
└── 材质功能：stretch fabric（高弹面料，便于跳跃）
```

**Fashionpedia标签**：`top` + `short length` + `sleeveless` + `color block` + `pleated` + `stretch`

---

#### 2. 女仆装（Maid Outfit）

```
基础品类：dress（连衣裙）或 blouse + skirt（分体）
├── 长度变体：midi length（中长裙，复古感）
├── 版型变体：a-line + fitted bodice（修身胸衣+外展裙摆）
├── 领型变体：peter pan collar（娃娃领）或 square neck（方领）
├── 袖型变体：puff sleeve（泡泡袖，甜美）或 short sleeve
├── 颜色图案：solid black/white（纯色黑白）+ contrast apron（撞色围裙）
└── 细节装饰：apron（围裙）+ bow（蝴蝶结）+ lace trim（蕾丝边）+ ruffle（荷叶边）
```

**Fashionpedia标签**：`dress` + `midi length` + `peter pan collar` + `puff sleeve` + `white` + `apron` + `bow` + `lace`

---

#### 3. 圣诞服（Christmas Outfit）

```
基础品类：dress / top / jumpsuit（视款式）
├── 长度变体：mini / short / midi（视性感程度）
├── 颜色图案：red/green/white（红绿白）+ striped（条纹）+ printed（图案：雪花/圣诞树/驯鹿）
├── 材质纹理：faux fur（人造毛）+ velvet（丝绒）+ knit（针织）
└── 细节装饰：fur trim（毛边）+ sequin（亮片）+ bow（蝴蝶结）+ bell（铃铛）
```

**Fashionpedia标签**：`dress` + `red` + `faux fur` + `sequined` + `fur trim`

---

#### 4. 睡衣（Sleepwear/Pajamas）

```
基础品类：underwear / dress（睡裙）或 top + pants（套装）
├── 长度变体：short / midi / maxi（睡裙款）或 cropped（长裤款）
├── 版型变体：loose fit（宽松）或 oversized
├── 领型变体：v-neck / scoop neck / button front（前开扣）
├── 袖型变体：short sleeve / long sleeve / sleeveless
├── 材质纹理：satin（缎面）+ lace（蕾丝）+ cotton（棉）+ modal（莫代尔）
└── 细节装饰：bow（系带蝴蝶结）+ ruffle（荷叶边）+ piping（滚边）
```

**Fashionpedia标签**：`underwear` + `loose fit` + `v-neck` + `satin` + `bow` + `piping`

---

#### 5. 布偶装（Mascot Costume / Fursuit）

```
基础品类：jumpsuit（连体衣）+ accessory（头套）
├── 版型变体：oversized（超大，覆盖全身）+ padded（填充，造型）
├── 材质纹理：faux fur（人造毛）+ fleece（摇粒绒）+ foam（泡沫填充）
├── 颜色图案：solid color（纯色块）+ animal print（动物纹，视角色）
└── 细节装饰：hood（连头套）+ tail（尾巴）+ paw gloves（爪形手套）
```

**Fashionpedia标签**：`jumpsuit` + `oversized` + `faux fur` + `hood` + `padded`

**注意**：这是**表演服装**，Fashionpedia的27个品类中没有专门的"costume"类，最接近的是`jumpsuit` + `accessory`

---

#### 6. 拘束服（Bondage Outfit / Straitjacket）

```
基础品类：无直接对应（功能特殊）
├── 版型变体：tight fit（紧身）+ bodycon（包身）
├── 材质纹理：leather（皮革）+ latex（乳胶）+ PVC
├── 闭合方式：buckle（皮带扣）+ strap（绑带）+ zipper（拉链）+ lock（锁扣）
├── 颜色图案：solid black（纯黑）为主
└── 结构特征：cross-over back（交叉背）+ crotch strap（胯带）+ long sleeve（长袖束缚）
```

**Fashionpedia标签**：**不完全匹配**，最接近的是`jacket` + `tight fit` + `leather` + `buckle` + `strap`

**分类困境**：这是**功能导向**的服装（限制活动），不属于日常穿着，Fashionpedia作为时尚数据集**未收录**此类。

---

#### 7. 露脐装（Crop Top）

```
基础品类：top / tank top / t-shirt
├── 长度变体：micro length / cropped length（超短，露脐）
├── 版型变体：slim fit（修身）或 tight fit（紧身）
├── 领型变体：crew neck / v-neck / scoop neck / off shoulder 等任意
├── 袖型变体：sleeveless / short sleeve / long sleeve 等任意
└── 搭配逻辑：必须配 high waist bottom（高腰下装）
```

**Fashionpedia标签**：`top` + `micro length` + `slim fit` + [任意领/袖]

**本质**：这是**长度变体**，不是独立品类。任何上衣（T恤、衬衫、毛衣）缩短到露脐长度，都可称为"露脐装"。

---

#### 8. 和服（Kimono）

```
基础品类：dress / robe（长袍）
├── 长度变体：maxi length / floor length（长至脚踝或地面）
├── 版型变体：straight fit（直筒，无收腰）+ wide sleeve（宽袖）
├── 领型变体：cross-over collar（交领右衽，中式/日式特征）
├── 袖型变体：wide sleeve（宽袖，袖与身连）+ long sleeve
├── 材质纹理：silk（丝绸）+ brocade（织锦）+ printed（印花）
└── 细节装饰：obi（宽腰带）+ sash（腰封）+ pleat（褶）+ tie（系带）
```

**Fashionpedia标签**：`dress` + `maxi length` + `straight fit` + `wide sleeve` + `cross-over collar` + `silk` + `obi`

**特殊地位**：和服是**民族服装**，有独立的文化体系和穿着规则，Fashionpedia将其归入`dress`大类，但实际应用中常作为**独立品类**处理。

## 三、分类规律总结

### 这些服装的命名逻辑

| 命名方式         | 例子              | Fashionpedia处理方式         |
| :----------- | :-------------- | :----------------------- |
| **场景/功能命名**  | 拉拉队服、女仆装、圣诞服、睡衣 | 映射到基础品类+属性组合             |
| **长度变体命名**   | 露脐装             | 基础品类+`micro length`属性    |
| **民族/文化独立**  | 和服              | 归入`dress`但需特殊标注          |
| **表演/角色扮演**  | 布偶装             | `jumpsuit`+`accessory`组合 |
| **亚文化/功能特殊** | 拘束服             | **未收录**，需自定义类别           |

### 实战分类公式

```
特殊服装 = 基础品类 + [关键变体1] + [关键变体2] + ... + [场景标签]

例：
拉拉队服 = top + short length + sleeveless + color block + pleated + [cheerleading]
女仆装 = dress + midi length + peter pan collar + puff sleeve + apron + bow + [maid]
露脐装 = top + micro length + slim fit + [crop]
和服 = dress + maxi length + straight fit + wide sleeve + cross-over collar + obi + [kimono]
```
