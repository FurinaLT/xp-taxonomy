## 服装衍生关系术语体系（汉化版）

### 核心概念层级

```
原型（原型/基型）
    ↓ 通过 变化方式 产生
变体（变体/变款）
    ↓ 通过 组合/叠加 产生
衍生款（衍生款/派生款）
    ↓ 通过 场景化/文化赋值 产生
风格化款式（风格款/定款）
```

## 一、变化方式术语（汉化）

| 术语     | 定义     | 操作对象            | 示例          |
| :----- | :----- | :-------------- | :---------- |
| **截短** | 减少长度   | 衣长、袖长、裤长        | T恤 → 露脐装    |
| **加长** | 增加长度   | 衣长、袖长、裤长        | 衬衫 → 衬衫裙    |
| **加宽** | 增加围度   | 下摆、袖口、裤脚        | 直筒裤 → 阔腿裤   |
| **收窄** | 减少围度   | 下摆、袖口、裤脚        | 直筒裤 → 锥形裤   |
| **去除** | 删除某部件  | 袖子、领子、裤腿        | 长袖 → 无袖     |
| **添加** | 增加新部件  | 背带、帽子、口袋        | 裤子 → 背带裤    |
| **连接** | 合并独立部件 | 上衣+裤子           | 衬衫+短裤 → 连体裤 |
| **分离** | 拆分连体结构 | 连衣裙 → 上衣+裙子     | 连衣裙 → 套装    |
| **开口** | 创造新的暴露 | 领口、背部、侧缝        | 圆领 → 深V领    |
| **闭合** | 减少暴露   | 无袖 → 短袖 → 长袖    | 吊带 → 罩衫     |
| **叠加** | 增加层次   | 假两件、拼接          | T恤 → 假两件T恤  |
| **替换** | 用A替代B  | 纽扣换拉链、圆领换V领     | 套头 → 前开扣    |
| **变形** | 改变结构形态 | 直筒 → A字、平领 → 翻领 | 衬衫领 → 青果领   |
| **换料** | 改变面料   | 棉→丝、牛仔→皮革       | 棉T恤 → 丝质T恤  |
| **赋图** | 添加视觉元素 | 印花、刺绣、条纹        | 白T恤 → 条纹T恤  |

## 二、衍生类型术语（汉化）

| 术语 | 定义 | 与原型的关系 | 示例 |
|:---|:---|:---|:---|
| **子型** | 原型下的细分 | 继承全部特征+特定限制 | 裤子→牛仔裤/西裤/运动裤 |
| **变体** | 单一维度变化 | 改变1个核心属性 | 长裤→短裤（长度变体） |
| **变型** | 多维度变化 | 改变2+个核心属性 | 衬衫→衬衫裙（长度+结构） |
| **混合型** | 两个原型的融合 | 50%+50%或主+次 | 衬衫裙（衬衫+裙子） |
| **衍生款** | 原型基础上的创新 | 保留识别度+新功能 | 卫衣→连帽卫衣→oversize卫衣 |
| **迭代款** | 时代/趋势更新 | 相同概念+当代审美 | 70年代喇叭裤→现代微喇裤 |
| **重构款** | 打破原结构重组 | 解构主义设计 | 西装→解构西装（无领/无扣） |
| **戏仿款** | 夸张/幽默化处理 | 放大特征或混搭 | 超大西装（Oversized Blazer） |
| **转用款** | 改变使用场景 | 内衣外穿、睡衣出街 | 睡衣→外穿衬衫 |
| **文化款** | 添加文化符号 | 民族元素、亚文化标志 | 普通外套→和服领外套 |

## 三、关系描述语法（汉化版）

### 基础句式

```
[原型] + [变化方式] → [衍生类型]: [新名称]

例：
T恤 + 截短 → 变体: 露脐装
衬衫 + 加长 → 变型: 衬衫裙
裤子 + 添加背带 → 变体: 背带裤
衬衫 + 裙子 + 融合 → 混合型: 衬衫裙
西装 + 加宽肩部 + 添加垫肩 → 迭代款: 权力西装
```

### 复杂句式（多步衍生）

```
[原型] 
    → [第一步变化] → [中间体]
    → [第二步变化] → [最终衍生款]

例：
西装原型
    → 加宽 + 垫肩 → 权力西装
    → 加长至大腿 → 西装大衣
    → 去除袖子 → 西装马甲
    → 变形领子为青果领 → 吸烟装夹克
```

## 四、实战：完整衍生链描述（汉化版）

### 案例：从"衬衫"到各种衍生

```
原型：衬衫
│
├── 长度维度 ──┬── 截短 → 露脐衬衫 [变体]
│              ├── 常规 → 标准衬衫 [原型]
│              └── 加长 → 衬衫裙 [变型]
│
├── 袖型维度 ──┬── 去除 → 无袖衬衫/背心 [变体]
│              ├── 缩短 → 短袖衬衫 [变体]
│              ├── 常规 → 长袖衬衫 [原型]
│              └── 变形 → 泡泡袖衬衫 [变体]
│
├── 结构维度 ──┬── 连接裤装 → 连体裤 [混合型]
│              ├── 叠加吊带 → 假两件衬衫 [衍生款]
│              └── 重构 → 解构衬衫 [重构款]
│
├── 开口维度 ──┬── 开口前襟 → 开衫式衬衫 [变体]
│              └── 闭合为套头 → 套头衬衫 [变体]
│
├── 版型维度 ──┬── 加宽 → 宽松衬衫/Oversized [变体]
│              └── 变形为收腰 → 衬衫式连衣裙 [变型]
│
└── 文化维度 ──┬── 添加中式立领 → 中式衬衫 [文化款]
               ├── 添加和服袖 → 和风衬衫 [文化款]
               └── 换料为丝绸+赋图刺绣 → 唐装 [文化款]
```

## 五、属性标签化表述（汉化版）

| 标签类型 | 格式 | 示例 |
|:---|:---|:---|
| 原型标签 | `基:[品类]` | `基:衬衫` |
| 变化标签 | `变:[方式]_[对象]` | `变:截短_长度`, `变:添加_帽`, `变:加宽_下摆` |
| 衍生类型 | `类:[类型]` | `类:变体`, `类:混合型`, `类:衍生款` |
| 结果名称 | `名:[通用名]` | `名:露脐装`, `名:衬衫裙` |

### 完整标签示例（汉化）

```
露脐装：
基:上衣 + 变:截短_长度 + 类:变体 + 名:露脐装

衬衫裙：
基:连衣裙 + 变:加长_长度 + 变:衬衫式门襟 + 类:混合型 + 名:衬衫裙

背带裤：
基:裤子 + 变:添加_背带 + 变:添加_护胸 + 类:变体 + 名:背带裤

连帽卫衣：
基:毛衣 + 变:添加_帽 + 变:长袖_袖型 + 类:衍生款 + 名:连帽卫衣
```

## 六、速查工具（汉化版）

### 变化方式速查表

| 你想表达... | 用这个词 | 符号 |
|:---|:---|:---|
| 变短 | 截短 | ↓长度 |
| 变长 | 加长 | ↑长度 |
| 变宽 | 加宽 | ↑围度 |
| 变窄 | 收窄 | ↓围度 |
| 加部件 | 添加 | +部件 |
| 减部件 | 去除 | -部件 |
| 合并 | 连接 | +品类 |
| 拆分 | 分离 | ÷品类 |
| 改变形态 | 变形 | ~形态 |
| 换材质 | 换料 | ⇄材质 |
| 加图案 | 赋图 | @图案 |
| 夸张化 | 夸张 | !特征 |

### 衍生类型速查表

| 变化程度 | 类型 | 识别点 |
|:---|:---|:---|
| 1个属性变 | 变体 | 容易认出原型 |
| 2-3个属性变 | 变型 | 需要联想原型 |
| 两个品类融合 | 混合型 | 一眼看出双重身份 |
| 功能场景变 | 衍生款 | 新用途但像原型 |
| 时代更新 | 迭代款 | 复古或未来感 |
| 结构打破 | 重构款 | 解构、不对称 |
| 文化符号加 | 文化款 | 民族/亚文化特征 |
