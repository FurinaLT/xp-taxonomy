# 一、你这套系统在本质上是什么

你现在做的不是“给衣服贴标签”，  
而是：

> 用一套规则，描述一件衣服是如何从另一件衣服“变出来”的。

就像生物进化一样：

- 原型 = 祖先
    
- 变化方式 = 变异
    
- 衍生款 = 新物种
    
- 风格化 = 被文化和场景塑造后的样子
    

所以一件衣服不是一个点，而是一条路径 🌿

---

# 二、核心结构（中文化）

你原本的层级可以这样理解：

```
原型
  ↓ 通过「变化方式」
变体
  ↓ 通过「组合/多次变化」
衍生形态
  ↓ 通过「文化与场景赋值」
风格化服装
```

换句话说：

**一件衣服 = 它从什么来 + 经历了哪些变化 + 被放进了什么文化**

# 三、变化方式（怎么变）

这是你体系里最重要的一层，我给你一份更“中文直觉”的版本：

|中文术语|含义|作用对象|例子|
|---|---|---|---|
|缩短|变短|衣长、袖长、裤长|T恤 → 露脐装|
|加长|变长|衣长、袖长|衬衫 → 衬衫裙|
|变宽|更蓬、更大|下摆、袖口、裤脚|直筒 → 阔腿|
|变窄|更收|裤脚、腰|直筒 → 锥形|
|去除|拿掉某部分|袖子、领子|长袖 → 无袖|
|添加|加零件|帽子、背带、口袋|裤子 → 背带裤|
|连接|两件变一件|上衣+裤子|衬衫+短裤 → 连体|
|分离|一件拆两件|连衣裙|连衣裙 → 上衣+裙|
|开口|开得更露|领口、背部|圆领 → 深V|
|闭合|更包|吊带 → 短袖||
|叠加|多一层|假两件、拼接|T恤 → 假两件|
|替换|A换成B|拉链、领型|套头 → 开襟|
|变形|改轮廓|A字、收腰|直筒 → A字|
|换材质|换布料|棉→丝、牛仔→皮|T恤 → 丝T|
|加图案|加视觉|印花、刺绣|白T → 花T|

这是一套“服装操作系统”。

# 四、衍生类型（它算变成了什么）

这是**人类认知层**，不是物理变化，而是“我们怎么看它”。

|中文名|含义|和原型的关系|
|---|---|---|
|子类型|同一大类里的细分|牛仔裤是裤子|
|变体|改了一个点|长袖 → 短袖|
|变型|改了好几个点|衬衫 → 衬衫裙|
|混合体|两个原型融合|衬衫+裙子|
|衍生款|基于原型扩展用途|卫衣 → 连帽卫衣|
|迭代款|时代更新|老喇叭裤 → 新喇叭裤|
|重构款|打破原结构|解构西装|
|戏仿款|夸张化|超大西装|
|功能转化|用途变了|睡衣外穿|
|文化赋值|加文化符号|中式、西式、亚文化|

# 五、你原本的“语法”，中文版

### 基础写法

```
[原型] + [变化方式] → [衍生类型] : [新名字]
```

例子：

```
T恤 + 缩短 → 变体 : 露脐装
衬衫 + 加长 → 变型 : 衬衫裙
裤子 + 添加背带 → 变体 : 背带裤
衬衫 + 裙子 + 融合 → 混合体 : 衬衫裙
```

---

### 多步变化链

```
西装
 → 加宽肩部 + 加垫肩 → 权力西装
 → 加长 → 西装大衣
 → 去掉袖子 → 西装马甲
 → 改领型 → 吸烟装夹克
```

这就是一条“进化路径”。
