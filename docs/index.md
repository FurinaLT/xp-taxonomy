# 前言

在二次元、虚构创作、角色设定与社区语境中，常见的标签往往同时混合了多种含义：  
有的描述生理结构，有的描述心理倾向，有的依赖文化语境，有的本身就是多个条件的压缩表达。  
当这些不同层级的概念被并列使用时，分类会迅速变得混乱，扩展也变得困难。

在结构上，本项目采用 **“主维度 + 附属词”** 的方式来描述复合概念：  
主维度用于回答“这个概念主要是什么”
附属词则从结构、表现、心理、行为、语境或叙事等不同角度，对其进行补充说明。

需要特别说明的是，本项目并不试图裁定“正确的理解”，  
也不以现实身份、个人认同或价值判断为目标。  
所有分类与编码，仅用于**结构化描述概念本身的构成方式**，以便于检索、组合、扩展与讨论。

简而言之：

> **词汇是给人使用的，结构是为了让系统不崩溃。**

如果这套体系能帮助读者更清楚地意识到：  
“我正在使用的这个词，其实同时涉及了哪些不同层面的含义”，  
那么这个项目的目的就已经达成了。

## 修改

与 V0.2 不同的是，**保留了基本词条，去除了复合词条**，并把基本词条修改为**词条**，增加了**附属词**用于衡量、分类和判断词条所包含的附属词有哪些。

例如 **黑瞳** 这个词的意思包含 **黑色、瞳孔颜色**，再次拆分和合并可以获得 **颜色、瞳孔** 这些词，一个是表现词另一个是部位词。表现词权重更大（主观），所以把他放到 外貌与视觉特征 而不是 本体/生物结构。
