## 关系

在摒弃了 [Tim2354](https://space.bilibili.com/360215933?spm_id_from=333.337.0.0) 的上下位分类法后，我们采用了更好的方法进行关系的展示。
使用 Markdown 列表进行制作，关系列表中有：**从属、衍生、影响**，三个维度，分别表示：

- **从属**：属于什么之下，类似于属于是什么的子级的关系
- **衍生**：衍生或是产生了什么，类似于与其他词条融合后产生的关系
- **影响**：影响了什么，类似于与其他词条融合后的关系

关系图可以对这种理解加深一步，并且可以发现更多的，关系列表中没有的关系。在下一节关系图中会详细讲解。

> 如果还不理解的可以理解为，与某个词条关联的程度，是这样递减的：从属、衍生、影响

## 关系图

采用 mkdocs 插件 graph 进行关系图的制作，程序会检查在文档内相互链接的文档，然后绘制箭头什么的。

如果一个A文档有一个**指向**B文档的链接，那么在关系图中就会显示出 A → B 这样的关系，如果C文档里面有一个指向A文档的链接，那么关系图中就会变成这样 C → A → B。
实际上并不会显示全部，而是某个文档**临近**的一个节点（文档）。你可以点击查看全部而查看全部文档的关系图，也就是 Full，但是会非常的卡顿，在后期随着数量变多也会非常的卡，**所以尽量不要点击 Full（查看全部节点关系图）**。

点击关系图选项中的**临近 Neihbors**选项可以查看与该节点临近一个节点的节点，我更推荐这个。在宽屏中，右屏的章节目录下方会自动显示关系图。

### (对于编写者)如何整理词条关系

在并不需要整理全部的词条指向关系的时候，编写可能会变得轻松，只需要看关系图就可以了。

思考一个词条指向某个词条的关系，一个箭头从起始（A）出发直到对应的词条（B），是直线也有指向性。这表明起始词条（A）对于终点词条（B）是这样的：**在从属关系上是子级**（A为B的子级）；**在影响层面上是对其进行影响**（A影响了B）；**在衍生层面上是由其衍生**（A衍生了B）。

所以根据词条和词条之间的指向感觉，进行指向评估之后就可以知道，怎么处理在哪一个词条中确定链接指向了。

## 新归类方法(第二期特有)

第二期项目把词条分成为三个大类：**产生性质、学术性质**。

- **产生性质**的词指的是在 **ACGN下产生的词汇**，或者是小众圈子产生的词汇在ACGN圈仍然适用的词汇，前者概括等级比后者高，避免其他性质词汇归类在此性质下的情况发生。
例如猫娘、futa可以归类在物种，萌袖等可以归类在服装。
- **学术性质**指的是在某一领域产生的**专用术语或者词汇**的词汇。例如斜方肌可以归类在医学-解剖学，三分裤可以归类在设计-服装。

**普遍性质分类已被移除**
以下是历史版本对本性质的说明

> **普遍性质**的词汇指的是**日常领域**，或者是民间流传的词汇，被大众所知，被字典所收录等词汇。例如丝袜、西装和牛仔裤可以分类在服装，西红柿和番茄可以分类在物品（或食物）。
> 
> 此外也有**跨界性质**的词汇，对于不同领域可以使用不同方式的解释方式的方法，在此情况下在词汇后加入括号+领域表示所在的性质。
